# Real-Time Task Tracking (Socket.IO)

## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB
- Socket.IO

## Features
- Real-time service/task status updates
- Live synchronization across multiple clients
- Socket.IO based event communication

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend/socket
npm install
npm run dev
